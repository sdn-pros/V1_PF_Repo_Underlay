#### Router BGP Device Configuration

```eos
!
router bgp 65101
   router-id 1.1.1.1
   neighbor CORE peer group
   neighbor CORE remote-as 65001
   neighbor 10.1.5.1 peer group CORE
   neighbor 10.1.5.3 peer group CORE
   !
   address-family ipv4
      network 1.1.1.1/32
      network 172.16.10.0/24
```
